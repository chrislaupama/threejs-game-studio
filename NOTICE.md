# Attribution And Research Notice

The distributed package is one self-contained coordinator with internal
reference chapters. The repositories named below are research or attribution
sources only; the installed package does not load or depend on them at runtime.

## Adapted MIT-licensed sources

Earlier versions of this repository incorporated and adapted
provider-independent game-development guidance and assets from:

- `mintdotgg/mint-threejs-skills`, commit
  `489a71a9d50e662f99b08810267d2fc6edc08219`, under its bundled MIT license.
- `majidmanzarpour/threejs-game-skills`, commit
  `8919acca095be36ea054c866a79997fca90e8b22`, under the MIT license reproduced
  below. Copyright (c) 2026 Majid Manzarpour.

Retained/adapted subject matter includes production sequencing, game design,
gameplay architecture, game feel, procedural modeling, rendering, technical
art, UI, debugging, QA, deterministic testing patterns, and the three reviewer
calibration images in `assets/scorecard-anchors/`.

Provider clients, credential probes, hosted generation, CDN runtime examples,
remote synchronization, analytics, and mandatory external-sourcing flows are
excluded. The operational package is written around project-owned or
user-supplied local files, procedural Three.js/browser content, and local
browser verification.

## Comparative research source

`CloudAI-X/threejs-skills` at commit
`b1c623076c661fc9b03dac19292e825a5d106823` was studied as a broad topic index
for Three.js fundamentals, geometry, materials, textures, lighting, loaders,
animation, interaction, shaders, and post-processing.

That repository does not contain a full `LICENSE` file at the studied commit;
its README contains only a short MIT statement. No substantial expression or
code was copied from it. Examples in this package were independently authored
and checked against official Three.js r185+ documentation, source, types, and
examples.
